## Overview

The OpenDESSEM wizard is an interactive command-line tool that guides users through:

1. System configuration (buses, submarkets, generators, loads)
2. Constraint selection and configuration
3. Solver options
4. Optimization and result display

Perfect for new users, testing, and quick prototyping.
